Command

ls is a command that allows you to view items in a directory.
Added from version: Konsole v1

Syntax: ls args [PATH]

args - arguments.
[PATH] is path to the directory, if it is not available, the current directory is used.

Arguments:
  • --n - shows element on new line
  • --info - shows additional information such as the date of creation, the size of the item
    • --edir - doesn't calculate size of directories
  • --dir - shows only directories
  • --file/--f - shows only files
  • --[EXTENSION] - shows only files with extension: [EXTENSION]
  • Sort:
    • --a-z - for alphabet
    • --cd - for date creation
    • --size - for size
