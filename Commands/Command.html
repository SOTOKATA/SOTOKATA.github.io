Commands

Here are all the commands available in Vonsole.

lsA command that shows all items in a directory
cd Allows you to move to another directory
info Allows you to find out information about the item
find Allows you to find out if the item is in the directory
run Allows you to launch the element
create Allows you to create directories and files
del Allows you to delete directories, files
copy Allows you to copy files, directories to a directory
cut Allows you to cut files, directories to a specific directory
for Allows you to repeat command
rename Allows you to change the file name, directory
clear Allows you to clear the screen
exit Exit the program
help List of all commands
