Structure [USER] <[PATH]> $: [USER_COMMAND]
  • [USER] - user name
  • <[PATH]> - path to correct directory
  • [USER_COMMAND] - user input
Commands
  • ls - list directory
  • cd - change directory
  • run - run/open item
  • create - create file/directory in correct directory
  • del - delete file/directory in current directory
  • copy - copy file/directory
  • cut - cut file/directory
  • find - find file/directory
  • info - information about file/directory
  • for - repeat command
  • help - list of all commands
  • clear - clear screen
  • exit - exit Konsole
