Konsole v1

Konsole v1 - перша версія Vonsole яка була переписана в версії Konsole v2

Structure [USER] <[PATH]> $: [USER_COMMAND]
  • [USER] - user name
  • <[PATH]> - path to correct directory
  • [USER_COMMAND] - user input
Commands
  • ls - list directory
  • cd - change directory
  • echo - echo the file/text
  • create - create file in currect directory
  • create[] - create files in current directory
  • del - delete file in current directory
  • del[] - delete files in current directory
  • mkdir - make directory
  • mkdir[] - make directories
  • copy - copy file
  • copy[] - copy files
  • dcopy - copy directory
  • dcopy[] - copy directories
  • cut - cut file
  • cut[] - cut files
  • dcut - cut directory
  • dcut[] - cut directories
  • find - find file/directory
  • info - information about file/directory
  • clear - clear screen
  • exit - exit Konsole
